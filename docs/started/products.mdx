# Productos

## Registra tus productos (opcional)

También puedes registrar tu inventario de productos en Facturapi para no enviar los mismos datos cada vez.

Para conocer todos los argumentos que puedes incluir en la llamada, consulta la [referencia del método Crear Producto](/api/#operation/createProduct).

<Tabs groupId="codeExamples">
<TabItem value="curl" label="cURL">

```bash
curl https://www.facturapi.io/v1/products \
  -H "Authorization: Bearer sk_test_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
        "description": "Ukelele",
        "product_key": "60131324",
        "price": 345.60,
        "sku": "ABC1234"
    }'
```

</TabItem>
<TabItem value="js" label="Node.js" default>

```javascript
const product = await facturapi.products.create({
  description: 'Ukelele',
  product_key: '60131324',
  price: 345.60,
  sku: 'ABC1234'
});
// save the product.id in your database
```

</TabItem>
<TabItem value="cs" label="C#">

```csharp
var product = await facturapi.Product.CreateAsync(new Dictionary<string, object>
{
  ["description"] = "Ukelele",
  ["product_key"] = "60131324",
  ["price"] = 345.60,
  ["sku"] = "ABC1234"
});
```

</TabItem>
<TabItem value="php" label="PHP">

```php
$product = array(
  "description" => "Ukelele",
  "product_key" => "4319150114",
  "price" => 345.60,
  "sku" => "ABC1234"

$new_product = $facturapi->Products->create( $product );
```
